Browser Automation Agent + Job Bot

An intelligent browser automation agent built with Python and Selenium/Playwright that scrapes job listings from job boards and automatically applies to them by filling out and submitting application forms.

Features


🔍 Job Scraping — Collects job listings from configured job boards based on search criteria (title, location, keywords, remote/on-site).
📝 Auto-Apply — Automatically fills and submits application forms (name, contact info, resume upload, cover letter, custom questions) using a stored applicant profile.
🤖 Browser Automation — Drives real browser sessions via Selenium or Playwright, handling dynamic pages, pop-ups, and multi-step forms.
⚙️ Configurable Rules — Filter jobs by keyword match, salary range, or exclusion list before applying.
🧠 Resume/Answer Matching — Fills free-text application questions using templated or AI-generated responses.
🔁 Retry & Error Handling — Gracefully handles failed page loads, CAPTCHAs, and unexpected form layouts.
